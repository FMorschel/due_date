Returns the next date that fits the day and the week.

  • If the current date - DateTime.day is less than the DateTime.month week, it's returned the same month with the DateTime.day being the day of the week.
  • If the current date - DateTime.day is greater than the DateTime.month week, it's returned the next month with the DateTime.day being the day of the week.

Implementation

@override
DateTime startDate(DateTime date) {
  final thisMonth = addMonths(date, 0);
  if (thisMonth.compareTo(date) >= 0) {
    return thisMonth;
  } else {
    return addMonths(date, 1);
  }
}
