DueDateTime<T extends Every>.now constructor

DueDateTime<T extends Every>.now({
  1. DateTime? limit,

Constructs a DueDateTime instance with current date and time in the local time zone. The DueDateTime.every property will be:

EveryDueDayMonth(day);

The limit will be passed to the Every instance if it is a LimitedEvery.

Implementation

factory DueDateTime.now({
  /// The limit for the operations.
  DateTime? limit,
}) {
  final now = clock.now();
  return DueDateTime.fromDate(
    now,
    every: EveryDueDayMonth(now.day) as T,
    limit: limit,
  );
}
