Returns a new DueDateTime instance with years years added to this.

If the current every is! EveryYear, the every used to create the new DueDateTime will be:

EveryDueDayMonth(day);

If sameEvery is true, keeps the current one. If is false, the every will change to the day of the generated date. And if the generated date is a leap year, and the month is February, the day will be set to 29 only if the current date is 29, else it will stay with the same day only changing the year.

EveryDueDayMonth(day);

Be careful when working with dates in local time.

Implementation

DueDateTime addYears(
  int years, {
  bool sameEvery = true,
}) {
  if (every is EveryYear) {
    return DueDateTime.fromDate(
      (every as EveryYear).addYears(this, years),
      every: sameEvery ? every : null,
    );
  } else {
    return DueDateTime.fromDate(
      EveryDueDayMonth(day).addYears(this, years),
      every: sameEvery ? every : null,
    );
  }
}
