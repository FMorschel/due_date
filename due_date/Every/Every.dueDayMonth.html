Class that processes DateTime so that the addMonths always returns the next month's with the DateTime.day as the dueDay clamped to fit in the length of the next month.

E.g.

  • If the dueDay is 31 and the next month end at 28, the addMonths will return the next month with the DateTime.day as 28.
  • If the dueDay is 31 and the next month end at 31, the addMonths will return the next month with the DateTime.day as 31.
  • If the dueDay is 15, the addMonths will return the next month with the DateTime.day as 15.
